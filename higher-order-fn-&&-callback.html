HigherOrderFunction && Callback

      * the function you’re passing as an argument is called a callback
        function and
      * the function you’re passing the callback function to is called a
        higher-order function.

Higher-Order Function

      * Is a function
      * Takes in a callback function as an argument
      * Returns a new function
      * The function it returns can invoke the original callback function that
        was passed in

Callback Function

      * It is a function used by another function to signal when there is a
      result ready.
      * It's a function that is passed as a parameter for another function
        where it is executed.

Before

        
          function add (x, y) {
            return x + y
          }
          ​
          function addFive (x, addReference) {
            return addReference(x, 5)
          }

After

      Because vocabulary is important, here’s the same code with the
      variables re-named to match the concepts they’re demonstrating.
    
      
        function add (x,y) {
          return x + y
        }
        ​
        function higherOrderFunction (x, callback) {
          return callback(x, 5)
        }
        ​
        higherOrderFunction(10, add)
