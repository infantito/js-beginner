Arguments

      Es un tipo especial de datos que se genera en el interior de cada función
      recogiendo todos los parámetros (argumentos) que se han enviado a la misma.
      

        
          El objeto arguments no es un Array. Es similar a un Array, pero no tiene
          ninguna propiedad Array excepto length. Por ejemplo, no tiene el método pop.
          Sin embargo este puede ser convertido a un Array real.
          Los arrow functions no exponen un objeto arguments [() =>]
        
      

      Particularidad del objeto arguments:
        [*]El objeto arguments no se crea si ya exite un parámentro formal con este nombre o
        si hemos declarado una variable homónima en el interior de la función
