Function Declaration vs Function Expression

      * F. Declaration:
        => These are load before any code is executed.
        => It's "defined" as soon as its surrounding function or script is
           executed (due to hoisting).

           e.g: function foo() { return any; }
      â€‹
      * F. Expression:
        => These are load only when the interpreter reaches that line of code.
        => It's only defined when that line is "reached". (due to hoisting)

        - Anonymous F.E:
          e.g: var foo = function() { return any; }

          - Named F.E:
          e.g: var foo = function foo() { return any; }
